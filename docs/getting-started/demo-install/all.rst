===================================================
Brightspot Demo Installation 
===================================================

The demo installation is a fast and easy way to create a personal version of Brightspot that runs on your computer. You can explore Brightspot through the included sample content from Inspire Confidence. The demo introduces you to the Dashboard and the Dari Developer Debug tool.

The demo installation uses Docker, a lightweight utility that simplifies the process of running Brightspot locally. Follow the installation procedure applicable to your operating system.

- For OS X, or the Windows 10 Professional or Enterprise 64-bit operating system, see :doc:`docker-platform`.

- For earlier versions of Windows, see :doc:`docker-toolbox`.
